# ORACLE

> Six tactical tools. Zero noise. Always watching.

## Tools
- 📸 Speed Camera Map — NJ speed & red-light cameras
- 👁 Attention Audit — Session tracking & deep work logging
- ♠️ Poker Equity Calc — Monte Carlo hand vs range equity
- ✈️ Flight Overhead — Aircraft above you via OpenSky Network
- 📻 Scanner Feed — Live NJ police/fire/EMS audio
- 🔮 Pre-Mortem Generator — AI plan stress-tester (Claude)

## Deploy
Static PWA. Add to iPhone home screen via Safari → Share → Add to Home Screen.

## Stack
Vanilla JS · Canvas API · OpenSky Network · Broadcastify · Anthropic API · Vercel
